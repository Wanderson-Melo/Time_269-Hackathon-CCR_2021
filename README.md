## Descrição do Projeto
O projeto Jornada do Primeiro Emprego é um aplicativo de vagas focado em facilitar a busca pelo primeiro emprego. Foi desenvolvido durante 2º Hackathon da CCR. 

## Tecnologias

- Javascript.
